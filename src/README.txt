###############################################################################

Como utilizar no Windows

###############################################################################

Este jogo depende do interpretador python 
http://www.python.org/ftp/python/2.5.4/python-2.5.4.msi
e da biblioteca pygame
http://pygame.org/ftp/pygame-1.9.1.win32-py2.5.msi

Para executar o jogo, basta instalar as dependências e executar o arquivo 
MathShooter.py (clicando duas vezes).

Caso o duplo click não funcione, basta abrir um terminal de linha de comando e
digitar: python MathShooter.py, estando na pasta do jogo,  ou 
python caminho\do\arquivo\MathShooter.py

###############################################################################

Como utilizar no Linux

###############################################################################

Basta instalar pygame pelo gernciador de pacotes, e executar o script via linha
de comando.
